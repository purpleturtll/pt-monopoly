# Monopoly

### How to run

```
# git clone https://github.com/purpleturtll/pt-monopoly

For development install nodemon:

# npm install -g nodemon

Server

# cd pt-monopoly
# npm install
# nodemon

Client

# cd pt-monopoly/client
# npm install
# cd ..
# npm run watch
```

### Visual Studio Code config for code completion

In file .vscode/jsconfig.json

```
{
    "compilerOptions": {
        "target": "es6"
    }
}
```
